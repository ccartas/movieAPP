# movieAPP

Android Native Application used for managing Movies

Functionalities:
- Login;
- Register;
- Add Facebook account;
- Movies Wish List;
- Movies Seen List;

Technologies:
- Android API 19;
- IMDB REST API;
- Side Menu (Gmail like)
- SQLITE DATABASE;
- ChartEngine library;
- Design Patterns: - Strategy;
                   - Builder;
                   - Factory;
                   - Singleton;
- Unit Testing;
